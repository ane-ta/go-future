* Синхронные вызовы (пользователь ждёт)

| Вызов                           | Макс. время | Обоснование                                                                                              |
| ------------------------------- | ----------- | -------------------------------------------------------------------------------------------------------- |
| `fraud.check()`                 | 100 мс      | Проверка правил в памяти (кеш), без обращения к БД. Исключение — редкие сложные кейсы (до 500 мс)        |
| `pricing.calculate()`           | 50 мс       | Чтение из кеша (Redis) предвычисленных тарифов и коэффициентов из потоков.                               |
| `payments.hold()`               | 3 сек       | Внешний API (Яндекс.Пэй). 95% запросов — 1 сек, 99% — 3 сек. Таймаут выставлен 5 сек                     |
| `payments.release()`            | 2 сек       | Отмена холда. Обычно быстрее, чем холд                                                                   |
| `driver.find()` (запуск поиска) | 50 мс       | Только постановка задачи в очередь. Сам поиск — асинхронный                                              |

* Асинхронные события (пользователь не ждёт)

| Событие                   | Публикует    | Когда                                              | Подписчики                                        |
| ------------------------- | ------------ | -------------------------------------------------- | ------------------------------------------------- |
| **`BookingCreated`**        | Оркестратор  | После успешного холда платежа                      | Driver, Notification                              |

| **`FindDriverRequested`** | Driver       | Получив `OrderCreated`, начинает поиск             | Driver (self) — для отслеживания состояния        |
| **`DriverProposed`**      | Driver       | Найден подходящий водитель, отправлено предложение | Оркестратор (ждёт принятия)                       |
| **`DriverAccepted`**      | Driver       | Водитель принял предложение                        | Оркестратор, Notification, Geo                    |
| **`DriverRejected`**      | Driver       | Водитель отказался                                 | Driver (продолжить поиск)                         |
| **`DriverAssigned`**      | Driver       | Водитель назначен                                  | Оркестратор (подтвердил заказ)                    |
| **`DriverSearchTimeout`** | Driver       | Поиск не дал результатов за N секунд               | Оркестратор (отменить заказ, вернуть деньги)      |
| **`BookingCancelled`**    | Оркестратор  | Заказ отменен, деньги разморожены                  | Driver                                            |

| **`RideStarted`**         | Driver       | Водитель начал поездку                             | Analytics, Notification                           |
| **`RideCompleted`**       | Driver       | Поездка завершена                                  | Payments (финальное списание), Payouts, Analytics |

| `SurgeFactorUpdated`      | Pricing      | Каждые N секунд (кешированное значение)            | Analytics, Geo (для поиска водителей с учётом спроса)   |
| `FraudAlert`              | Fraud        | При обнаружении подозрительной транзакции (редко)  | Analytics, Notification (уведомить службу безопасности) |

| **`PaymentHeld`**         | Payments     | Деньги заблокированы                               | Оркестратор                                       |
| **`PaymentReleased`**     | Payments     | Деньги возвращены                                  | Оркестратор                                       |
| **`PaymentCharged`**      | Payments     | Финальная сумма списана                            | Analytics                                         |

| **`PayoutCompleted`**     | Payouts      | Выплата водителю проведена                         | Analytics                                         |

| **`LocationUpdated`**     | Driver       | Каждые 3-5 секунд                                  | Geo, Analytics                                    |

| **`NotificationSent`**    | Notification | Push отправлен                                     | Analytics                                         |
