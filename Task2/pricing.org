* Ценообразование
**Подход:** Гибридный.
- Базовая цена рассчитывается раз в 5–10 секунд на основе потоковых данных (Flink)
- Результат кладётся в Redis
- Синхронный вызов `pricing.calculate()` читает из Redis (50 мс)

**Почему не полностью потоковый:**  
Слишком высокая задержка для UX такси (200–500 мс).

**Почему не чистое предвычисление:**  
Не отражает резкие изменения спроса (концерт, дождь).
